DOM, CSSOM, and Render Tree

The browser turns HTML into the DOM — a family tree of page elements: 

html
├── head
└── body
    ├── h1
    ├── img
    └── button

It reads CSS and makes the CSSOM (style tree).

Then it combines DOM + CSSOM into the Render Tree to know what to show.

Layout (Reflow) and Paint (Repaint)

Layout is like placing furniture in a room — the browser figures out where each element goes.

Paint is coloring and drawing pixels on the screen.

If the page changes, the browser might have to do layout and paint again, which can slow things down.
