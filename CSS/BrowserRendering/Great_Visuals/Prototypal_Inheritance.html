üå≥ Prototypal Inheritance

Imagine toys can borrow tricks from other toys! This is prototypal inheritance.

Look at this simple family tree of toys:

Object.prototype
‚û°Ô∏è

Root prototype with basic shared tricks.

// Dog inherits tricks from Animal, which inherits from Object
const dog = {
  bark() {
    console.log("Woof!");
  }
};
const animal = {
  eat() {
    console.log("Eating food!");
  }
};
Object.setPrototypeOf(dog, animal);
dog.bark(); // Woof!
dog.eat();  // Eating food! (inherited)
