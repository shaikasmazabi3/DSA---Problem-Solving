The @layer rule helps you organize these layers to keep styles neat and clear.

  body { background: white; }
}
@layer theme {
  body { background: lightblue; }
}

Performance & Wrap-Up üöÄ

Fast webpages make users happy! Let's meet some important speed friends:

  • TTFB - Time To First Byte: How fast your browser gets the first data from the server.
  • FCP - First Contentful Paint: When the first text or image appears.
  • LCP - Largest Contentful Paint: When the biggest part of the page loads.
  • TTI - Time To Interactive: When your page is ready to use.
  • CLS - Cumulative Layout Shift: How much things jump around on your page.
