How Does a Browser Load a Website?

Imagine the browser is a mailman who fetches a book (website) from a big library called the internet.

  1. You type amazon.com.
  2. The browser asks the internet phone book (DNS) where Amazon lives.
  3. Browser sends a letter (HTTP request) to Amazon’s server.
  4. Amazon sends back the main book (HTML file).
  5. Browser reads the book and sees notes to get styles (CSS), brain (JS), and pictures (assets).
  6. Browser fetches those and builds the website you see!

HTML, CSS, JavaScript, and Assets

HTML is the skeleton (structure):

<h1>Welcome to Amazon</h1>
<img src="logo.png" alt="Amazon Logo">

CSS is the clothes (styles):

h1 {
  color: orange;
  font-size: 24px;
}

JavaScript is the brain (interactivity):

document.querySelector('button').onclick = () => alert('Added to cart!');

Assets are decorations like images and videos.
