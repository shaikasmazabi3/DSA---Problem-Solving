Lazy Loading: Load Only What You See

Imagine a big coloring book with 100 pages but you only color the first 5 pages now.

Lazy loading means the browser only loads images or content when you scroll to see them.

Image not loaded yet

Without lazy loading, the browser loads all images at once, making the page slow.

Caching: Remembering to Save Time

When you visit a website, your browser saves some files in a secret box called cache.

Next time, the browser uses those saved files instead of asking the server again, making loading faster.

Cache

Think of it as your browser remembering your favorite book so you don’t have to borrow it again.
